# Olist Customer Analysis

## Overview
This project explores customer behavior and retention using the Olist e-commerce dataset.

## Done with
- DBeaver
- PostgreSQL

## Key Questions
- How does order volume change over time?
- Do customers return after their first purchase?
- What are retention patterns across cohorts?

## Key Takeaways
- Order growth over years is misleading due to incomplete 2016 data
- Business shows steady growth with seasonal peaks (Nov–Jan)
- ~97% of customers make only one purchase
- Retention drops to <1% after first month, indicating mostly one-time purchase behavior
- Cohort Analysis - across all cohorts: 90% of users make only one purchase.
